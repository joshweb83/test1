# 든이의 계산기 (Deun's Calculator)

## 🧮 프로젝트 소개
윈도우 기본 계산기와 유사한 웹 계산기입니다.

## ✨ 주요 기능
- 기본 사칙연산 (+, -, ×, ÷)
- 천단위 컴마 표시 (1,000, 12,345 등)
- 키보드 입력 지원
- 반응형 디자인 (모바일 지원)
- 윈도우 계산기 스타일 UI
